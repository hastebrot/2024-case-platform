# 2024-job-platform

**usage:**

- `❯ git clone https://github.com/hastebrot/2024-job-platform/`
- `❯ cd 2024-job-platform/sources/platform-app/`
- `❯ bun install`
- `❯ bun run dev --port 5678`
- `❯ open -a safari --url "http://localhost:5678/"`
